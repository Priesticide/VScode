색상값 참조 사이트 colorpicker

  • 16진수로 테스트
  • rgb로 테스트
  • rgba로 테스트
  • hsl로 테스트
  • hsla로 테스트

text-decoration 속성 : 글자의 밑줄, 취소선 등을 표현하는 속성

선택자 { text-decoration : none | underline | overline | line-through }
  • 밑줄 표현하기
  • 가운데 취소선 표현하기
  • 윗줄표현하기
  • 하이퍼 링크 표현하기
