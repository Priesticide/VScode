선택자란?
특정한 HTML 태그 요소를 선택할 떄 사용하는 표기 방식
원하는 태그를 선택하여 원하는 스타일을 각각 적용할 때 쓰인다.

모든 요소를 선택하는 선택자 : *

HTML 안의 모든 요소를 선택할 때 사용한다.

  • 테스트1
  • 테스트2
  • 테스트3

태그 선택자

HTML 문서 내의 동일한 태그 모두에게 같은 스타일을 적용할 때 사용한다.

        p { color : blue; } 와 같은 형태로 사용한다.

아이디 선택자

HTML문서 내의 선택한 아이디에 해당하는 스타일을 별도로 적용할 떄 사용하는 선택자

        #아이디명 { color : green; }

클래스 선택자

HTML 문서 내의 특정 그룹을 짓는 요소들을 하나의 스타일로 적용할 때 사용한다. 

 .클래스명 { color : purple; }
