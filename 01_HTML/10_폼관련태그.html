text관련 input태그

type = "text"

한 줄 짜리 텍스트를 작성할 수 있는 타입

type= "password"

비밀번호를 입력할 수 있는 타입

type = "search"

검색 전용 박스를 제공한다.

type = "email"

입력한 정보가 이메일이 맞는지 검사하는 타입

type = "tel"

입력한 정보가 연락처인지 확인하는 타입

type = "url"

입력한 값을 value값과 합하여 웹 문서를 호출하는타입

type = "submit/reset"

  

숫자와 관련된 input 태그

type = "number"

입력창에 스핀형태로 표현되나, 웹 브라우저에서 따라서 슬라이드 형태로 표현되는 경우도 있다.

type = "range"

슬라이드 바를 통해 숫자를 지정할 수 있다.

날짜와 관련된 input 태그

type = "date | month | week | time | datetime-local"
